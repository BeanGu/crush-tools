CRUSH (Custom Reporting Utilities for SHell) is a collection of tools for processing delimited-text data from the command line or in shell scripts.

For help getting started using CRUSH, or to see a demo of what it can do, try the CrushTutorial.  For a list of the utilities provided in CRUSH and links to their documentation, see the UserDocs.  Or see ApplicationDevelopmentWithCrush for a detailed look at writing applications using the CRUSH toolkit.

Join the CRUSH discussion group at http://groups.google.com/group/crush-tools

## Contributing ##

Want to help improve CRUSH?  There are plenty of ways you can help.

  * Report bugs by filing an [issue](http://code.google.com/p/crush-tools/issues/list).
  * Offer feedback and discuss feature requests on the [discussion group](http://groups.google.com/group/crush-tools).
  * Contribute source code.  [Check out](http://code.google.com/p/crush-tools/source/checkout) the sourcecode, read [the development guidelines](http://code.google.com/p/crush-tools/source/browse/trunk/HACKING), be sure to sign the [CLA](http://code.google.com/legal/individual-cla-v1.0.html), and post your patches to the [discussion group](http://groups.google.com/group/crush-tools).  If you want to help but don't know what to work on, the "Issues" tab above may contain items that interest you.
