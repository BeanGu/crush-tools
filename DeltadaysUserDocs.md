# Usage #

`deltadays <options> [start] [end]`



# Options #
|**Short Opt**|**Long Opt**|**Argument**|**Description**|
|:------------|:-----------|:-----------|:--------------|
|-h|--help|  |print this message and exit|
|-s|--start|

&lt;start&gt;

 |start date of range|
|-e|--end|

&lt;end&gt;

 |end date of range|
|-f|--format|

&lt;format&gt;

 |date format to use (default: YYYY-MM-DD)|
