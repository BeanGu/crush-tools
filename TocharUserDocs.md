# Usage #

`tochar <options> <N>`

N may be decimal, octal (with leading 0), or hexadecimal (with leading 0x)

# Options #
|**Short Opt**|**Long Opt**|**Argument**|**Description**|
|:------------|:-----------|:-----------|:--------------|
|-h|--help|  |print this message and exit|
